# 스킬: 미팅별 준비자료 (skill-meeting-prep)

> 1~3차 미팅 전 진우님이 빠르게 준비할 수 있도록 체크리스트 및 초안 제공

---

## 1차 미팅 준비

**트리거:** "○○ 1차 미팅 준비"

### 사전 확인 체크리스트
```
□ 고객 사전 정보 확인 (구글폼 인테이크)
  - 이름 / 연령대 / 관심 영역
  - 희망 미팅 형태

□ 준비할 자료
  - 재무상태표 작성 양식 (빈 템플릿)
  - 현금흐름표 양식
  - 보험 효율성 체크 양식

□ 미팅 중 다룰 항목
  □ 내 업무 소개 (5분)
  □ 현금흐름 파악
  □ 재무상태표 작성
  □ 보험 현황 및 효율성 체크
  □ 재무목표 설정 (목표명 / 금액 / 기간)
  □ 투자원칙 안내
  □ 2차 미팅 일정 확정
  □ To-do 리스트 전달
  □ 소개 요청

□ 미팅 후
  □ 상담일지 녹음
  □ 텍스트 변환 → 노션 저장
```

---

## 2차 미팅 준비

**트리거:** "○○ 2차 미팅 준비"

### 상담일지 요약 기반 준비 포인트
```
□ 노션에서 ○○ 고객 상담일지 확인
□ 재무목표 재확인 (목표 1, 2, 3)
□ 월 저축 가능액 확인

□ 상품 비교표 준비 (skill-product-compare 참조)
  □ 은행: 적금/예금 옵션
  □ 증권: ISA/IRP/연금저축 ETF 포트폴리오
  □ 보험: 보장성 + 저축성 조합

□ 미팅 중 다룰 항목
  □ 1차 미팅 내용 리마인드
  □ 목표별 상품군 설명
  □ 대략적 금액 조율
  □ 부담감 제거 대화
  □ 3차 미팅 일정 확정
  □ To-do 리스트 전달
  □ 소개 요청

□ 미팅 후
  □ 이탈방지 D+1,2,3 시퀀스 준비 (skill-dropout-prevention 참조)
```

---

## 3차 미팅 준비

**트리거:** "○○ 3차 미팅 준비"

### 포트폴리오 북 완성 체크
```
□ 포트폴리오 북 초안 완성 (skill-portfolio-book 참조)
  □ 섹션1: 재무현황 요약
  □ 섹션2: SWOT 분석
  □ 섹션3: 재무목표 타임라인
  □ 섹션4: 포트폴리오 구성
  □ 섹션5: 상품별 설명 요약
  □ 섹션6: 앞으로의 로드맵

□ 당일 ETF 포트폴리오 시세 확인 및 업데이트
□ 인쇄 준비 (컬러/흑백 확인)

□ 미팅 중 다룰 항목
  □ 1,2차 미팅 리마인드
  □ 포트폴리오 북 함께 리뷰
  □ 금융상품 계약 진행
  □ 계약 서류 준비
  □ 소개 요청 (구조화 스크립트 활용)
```
