# SOP: 신규 고객 온보딩

---

## 트리거
- consult.html에서 상담 예약 완료
- Google Forms 인테이크 제출

---

## 단계별 프로세스

### Step 1. 인테이크 수신 확인 (자동)
- Google Forms 제출 → GAS → Telegram 알림 수신
- Notion CRM에 자동 등록 확인

### Step 2. 상담 전 준비 (진우님)
- [ ] Notion CRM에서 고객 인테이크 정보 확인
- [ ] 고객 세그먼트 파악 (나이/직업/관심 상품)
- [ ] 맞춤 질문 2-3개 준비

### Step 3. 상담 진행
- Clova Note 녹취 시작
- 체크리스트: 니즈 파악 → 현황 진단 → 솔루션 제안 → 다음 단계 합의

### Step 4. 상담 후 처리 (자동 + 수동)
- [ ] Clova Note → 텍스트 추출
- [ ] 상담 분석 파이프라인 실행 (자동)
- [ ] 카카오 메시지 초안 검토 → 발송 (진우님 확인 후)
- [ ] Notion CRM 상태 업데이트

### Step 5. 팔로업 (3일 이내)
- [ ] 상담 내용 요약 카카오 발송
- [ ] 맞춤 콘텐츠/자료 공유
- [ ] 다음 상담 또는 계약 일정 제안

## 고객 세그먼트별 주요 관심 상품

| 세그먼트 | 주요 관심 | 우선 추천 |
|---|---|---|
| 30대 맞벌이 직장인 | 세금 절약, 노후 준비 | IRP + ISA + 연금보험 |
| 20대 직장인 초년 | 종잣돈 마련 | ISA + 적립식 펀드 |
| 40대 자산가 | 절세 + 수익 | 변액연금 + 해외주식 |
