# 스킬: 포트폴리오 북 생성 (skill-portfolio-book)

> 3차 미팅 전 고객에게 전달하는 개인 맞춤 책자 초안 생성

---

## 트리거
진우님: "○○님 포트폴리오 북 만들어줘"

---

## 필요 입력값
아래 정보를 진우님이 제공하거나 노션 상담일지에서 가져옴:

```
- 고객명 (이니셜 또는 닉네임)
- 나이 / 직업 / 가족구성
- 월 수입 / 월 지출 / 현재 저축액
- 현재 보험 가입 현황
- 재무목표 (목표1: 내용/금액/기간, 목표2: ...)
- 월 저축 가능액
- 투자 성향 (안정형/중립형/성장형)
- 특이사항
```

---

## 출력 구조 (6섹션)

### 섹션 1. 재무현황 요약
```
[고객명]님의 현재 재무 현황

💰 월 현금흐름
- 수입: ○○만원
- 지출: ○○만원
- 순저축 가능액: ○○만원

📊 자산 현황
- 금융자산: ○○
- 부채: ○○
- 순자산: ○○

🛡 보험 현황
- 가입 보험: ○○
- 효율성 평가: ○○ (개선 필요 항목)
```

### 섹션 2. SWOT 분석
```
강점 (Strength)
- 안정적인 직장 소득
- (고객 상황에 맞게)

약점 (Weakness)
- 보험 공백 구간 존재
- (고객 상황에 맞게)

기회 (Opportunity)
- 30대 초반 복리 최대 활용 시점
- (고객 상황에 맞게)

위협 (Threat)
- 인플레이션 대비 실질 수익률 관리 필요
- (고객 상황에 맞게)
```

### 섹션 3. 재무목표 타임라인
```
목표 1: [목표명]
- 목표금액: ○○만원
- 달성 시점: ○○년 후 (○○세)
- 월 필요 저축: ○○만원
- 추천 수단: ○○

목표 2: [목표명]
(동일 구조 반복)
```

### 섹션 4. 포트폴리오 구성
```
월 저축액 ○○만원 배분안

은행 (○○만원)
- 상품명 / 금리 / 기간 / 목적

증권 (○○만원)
- ISA / IRP / 연금저축 / ETF 구성

보험 (○○만원)
- 보장성: 상품명 / 월납 / 보장내용
- 저축성: 상품명 / 월납 / 수익 구조
```

### 섹션 5. 상품별 설명 요약
각 포트폴리오 상품마다:
```
[상품명]
- 특징: 1줄 요약
- 이 고객에게 맞는 이유: 2줄
- 주의사항: 1줄
```

### 섹션 6. 앞으로의 로드맵
```
계약 후 3개월: ○○
계약 후 6개월: ○○
1년 후: ○○
```

---

## 작성 원칙

1. **숫자 중심** — 추상적 표현 없이 금액/기간/수익률 명시
2. **고객 언어 사용** — 미팅에서 나온 표현 그대로 반영
3. **보험 거부감 완화** — 보험 섹션은 "보장"보다 "재무 안전망" 프레이밍
4. **확인 필요 표시** — 실제 상품 수치는 반드시 ⚠️ 확인필요 표기
5. **초안임을 명시** — 모든 출력 마지막에 "✅ 진우님 검토 후 수치 확인 필요"
