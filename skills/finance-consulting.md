## 고객 재무 컨텍스트 (JW Financial 핵심 타겟)

```
핵심 타겟: 30대 맞벌이 직장인
  - 소득: 부부 합산 연 8,000~1.5억
  - 자산: 부동산(전/월세 or 내 집) + 예적금 + 퇴직연금
  - 부채: 주택담보대출, 전세대출
  - 니즈: 세금 절약, 노후 준비, 자녀교육, 내 집 마련

주력 상품군:
  - 변액연금 (10년납) — 수익 90%↑
  - 보장성보험
  - IRP (퇴직연금 추가납입)
  - ISA (비과세 종합저축)
  - 연금저축펀드
```

## FRAMEWORK 1 — Damodaran 밸류에이션 사고 (고객 재무 진단)

**핵심 원칙:** "숫자에 스토리를 붙이고, 스토리에 숫자를 붙여라"

Damodaran은 기업 가치를 분석하지만, 그 방법론은 **고객 재무 진단**에 그대로 적용된다.  
"이 고객의 현재 재무 상태가 목표 달성에 충분한가?"

### 고객 재무 진단 5단계

```
1. 현금흐름 파악 (Cash Flow)
   - 월 수입 (고정 + 변동)
   - 월 지출 (고정: 대출/보험/저축 / 변동: 생활비)
   - 순 현금흐름 = 저축 여력

2. 자산-부채 구조 (Balance Sheet)
   - 자산: 부동산 / 금융자산 / 퇴직연금 / 보험 해지환급금
   - 부채: 주담대 / 전세대출 / 기타
   - 순자산 = 자산 - 부채

3. 목표 역산 (Target Valuation)
   - 목표: "63세부터 월 300만원 수령"
   - 역산: 지금부터 얼마를 어떻게 넣어야 하는가
   - 도구: 복리 시뮬레이션, 연금 수령액 계산

4. 갭 분석 (Gap Analysis)
   - 현재 궤도 vs 목표 궤도 차이
   - "지금 이대로면 63세에 얼마 있을까?"
   - "목표 달성하려면 월 얼마 더 필요한가?"

5. 포트폴리오 설계 (Portfolio Design)
   - 갭을 메우는 상품 조합 설계
   - 세제혜택 우선 활용 (IRP → ISA → 연금저축 → 변액)
   - 리스크 허용 범위 맞춤 자산배분
```

### 포트폴리오 북 6섹션 구조
```
Section 1: 재무현황 (현금흐름 + 순자산)
Section 2: SWOT 분석 (강점/약점/기회/위협)
Section 3: 재무 목표 (숫자로 표현된 구체적 목표)
Section 4: 포트폴리오 설계 (상품별 역할 + 금액)
Section 5: 상품 설명 (각 상품 왜 이게 맞는지)
Section 6: 실행 로드맵 (월별/연도별 체크포인트)
```

## FRAMEWORK 2 — Howard Marks 리스크 & 사이클 (매크로 분석)

**핵심 원칙:** "리스크는 나쁜 결과가 아니다. 예상치 못한 결과다."

핀사이트랩스 매크로 리포트와 고객 상담 시 시장 상황 설명에 적용.

### 사이클 포지션 판단 프레임

```
현재 사이클 위치 파악:
  ↑ 상승 사이클: 낙관론 팽배, 리스크 과소평가, 가격 고평가
  → 고점 근처: 방어적 포지션, 변동성 대비
  ↓ 하락 사이클: 비관론 팽배, 리스크 과대평가, 가격 저평가
  → 저점 근처: 공격적 포지션, 장기 매수 기회

신호 체크리스트:
  □ 금리 방향성 (인상/동결/인하 사이클)
  □ 신용 스프레드 (확대 = 리스크오프)
  □ 투자 심리 지표 (공포탐욕지수, VIX)
  □ 유동성 (M2, Fed 자산)
  □ 실물 경기 (PMI, 고용, 소비)
```

### 고객 상담 적용 — 시장 상황 설명 원칙
```
금지: "지금 시장이 안 좋아서 투자하면 안 돼요"
금지: "지금이 딱 좋은 타이밍이에요" (확신 금지)

권장: "현재 사이클 위치는 이렇고, 그래서 이 상품은 이런 역할을 합니다"
권장: "장기 관점에서 이 구조가 유리한 이유는..."
권장: "리스크는 이것이고, 대응 방법은 이렇습니다"
```

### 핀사이트랩스 매크로 리포트 구조
```
1. 현재 상황 (What)
   — 이번 주/이번 달 주요 지표, 이벤트 요약

2. 해석 (Why)
   — 왜 이런 결과가 나왔는가 (인과관계)

3. 사이클 포지션 (Where)
   — 현재 우리가 사이클 어디쯤 있는가

4. 시사점 (So What)
   — 개인 투자자/직장인에게 실질적 의미

5. 액션 포인트 (Now What)
   — 지금 할 것 / 지켜볼 것 / 피할 것
```

## FRAMEWORK 3 — Buffett + Munger (자본 배분 + 멘탈 모델)

**Buffett 핵심:** "훌륭한 사업을 적정한 가격에 사고, 오래 보유하라"
**Munger 핵심:** "거꾸로 생각하라. 실패를 피하면 성공이 온다"

### 자본 배분 원칙 → 고객 적용

```
Buffett 원칙 1: 이해할 수 있는 것에만 투자
  → 고객에게: "이 상품 구조를 완전히 이해하고 가입하세요"
  → 부장에게: 상품 설명은 반드시 비유로 시작

Buffett 원칙 2: 안전마진 (Margin of Safety)
  → 고객에게: "최악의 경우를 가정하고 설계합니다"
  → 부장에게: 시뮬레이션은 보수적 수익률 기준 (낙관 x)

Buffett 원칙 3: 복리의 힘 + 시간
  → 고객에게: "10년, 20년 후 차이를 숫자로 보여드립니다"
  → 부장에게: 시뮬레이션엔 반드시 연도별 누적 그래프 포함

Munger: 역산 사고 (Inversion)
  → "어떻게 하면 노후가 망할까?" → 반대로 하면 성공
  → 고객 상담에 활용: "이렇게 하면 실패하는 케이스를 먼저 보여드릴게요"
```

### 멘탈 모델 — 고객 설명에 쓰는 비유 라이브러리

```
복리:
  "눈덩이를 굴리는 것과 같아요. 처음엔 작지만 경사면이 길수록
   나중엔 기하급수적으로 커집니다"

분산투자:
  "한 바구니에 달걀을 다 담지 않는 거예요.
   바구니 하나가 떨어져도 전부 깨지지 않도록"

보험의 역할:
  "보험은 수익 상품이 아니에요. 소방서예요.
   불이 안 나면 쓸 일 없지만, 불 났을 때 없으면 끝납니다"

IRP 세액공제:
  "월급 300만원 받는데 나라에서 '이 통장에 넣으면
   연말에 66만원 돌려드릴게요' 하는 거예요"

변액연금 10년납:
  "10년 동안 씨앗 심고, 그다음은 나무가 자라는 걸 지켜보는 구조예요"
```

## 시뮬레이션 작업 원칙

```
1. 수익률 기준:
   - 보수적: 연 3~4% (원금보장형 참고)
   - 중립: 연 5~6% (균형 포트폴리오)
   - 낙관: 연 7~8% (주식형, 장기)
   → 반드시 3가지 시나리오 제시, 중립 기준 강조

2. 필수 표기:
   - ⚠️ 실제 수익률은 시장 상황에 따라 다를 수 있습니다
   - 세후 수령액 기준으로 계산 명시

3. 비교 구조:
   - "지금 시작 vs 5년 후 시작" 비교
   - "이 상품 조합 vs 예금만" 비교
   → 갭을 숫자로 보여줘야 결정이 쉬워진다
```
