# 스킬: 계약 후 온보딩 시퀀스 (skill-onboarding-sequence)

> 계약 후 90일간 자동 터치포인트 4회
> 현재 계약 후 대응이 거의 없는 문제 해결

---

## 트리거
진우님: "계약완료 ○○"

## D+0 환영 메시지 (계약 당일)

```
[고객명]님, 오늘 함께해주셔서 진심으로 감사드립니다 🙏

앞으로 [고객명]님의 재무 파트너로서
든든하게 옆에 있겠습니다.

오늘 계약하신 내용 간단히 정리해드릴게요:

📋 계약 요약
- [상품1]: ○○만원/월
- [상품2]: ○○만원/월
- 총 월 저축액: ○○만원

📅 앞으로 일정
- 3개월 후: 첫 관리미팅 예정
- 매주: 주간 시장 리포트 발송

언제든 궁금하신 점은 편하게 연락 주세요 😊

남진우 드림
✅ 검토 후 발송해주세요
```

---

## D+7 첫 체크인 (1주일 후)

```
[고객명]님, 안녕하세요!
계약하신 지 벌써 일주일이 됐네요 😊

가입하신 상품들 잘 정착되고 계신가요?
혹시 첫 납입 관련해서 궁금하신 점은 없으셨나요?

작은 것이라도 불편하거나 궁금하신 게 있으시면
편하게 말씀해 주세요.

남진우 드림
✅ 검토 후 발송해주세요
```

---

## D+30 1개월 리뷰 (한 달 후)

```
[고객명]님, 벌써 한 달이 지났네요!

이번 달 간단한 현황 공유드려요 📊

[관련 시장 동향 1-2줄 - 모닝브리핑 참조]

[고객명]님 포트폴리오와 관련해서는
크게 변동 사항 없이 계획대로 진행 중입니다.

다음 달에도 좋은 소식 가져오겠습니다 😊

남진우 드림
✅ 시장 동향은 모닝브리핑 내용으로 업데이트 필요
```

---

## D+90 분기 체크 (3개월 후)

```
[고객명]님, 안녕하세요!
어느새 3개월이 됐습니다 🙂

이제 슬슬 첫 관리미팅 일정 잡아볼까요?

지난 3개월 간 포트폴리오 현황 점검하고
새로 생긴 재무 변화나 목표도 함께 이야기 나누면 좋을 것 같아요.

아래 링크에서 편한 시간 선택해 주시면 됩니다 📅
→ [캘린더 링크]

남진우 드림
✅ 캘린더 링크 삽입 필요
```

---

## 작성 원칙

1. **부담 없는 톤** — 영업 느낌 없이 진짜 파트너처럼
2. **짧게** — 각 메시지 5줄 이내
3. **다음 단계 1개만** — CTA는 하나로 집중
4. **개인화** — 고객명, 계약 상품명 반드시 포함
