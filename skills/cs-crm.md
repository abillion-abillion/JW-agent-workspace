## FRAMEWORK 1 — Delivering Happiness (Tony Hsieh)

**핵심 원칙:** "고객을 감동시켜라. 기대를 넘어서는 순간 팬이 된다"

```
WOW 경험 설계:
  기대치: 고객이 자산관리사에게 기대하는 것
    → 연락 잘 됨, 정보 제공, 계약 처리
  
  JW Financial WOW 포인트:
    → 시장 급변 시 먼저 연락 (고객이 묻기 전에)
    → 생일/기념일 기억
    → D+30 경제 이슈 리마인드
    → 소개 시 Give 먼저 (재무진단 보고서 등)

서비스 문화 원칙:
  - 고객이 연락하기 전에 내가 먼저
  - 나쁜 소식일수록 빨리 투명하게
  - 문제 해결보다 관계 유지가 우선
```

## FRAMEWORK 2 — Never Lose a Customer (Joey Coleman)

**핵심 원칙:** "첫 100일이 평생 고객을 만든다"

### 전체 고객 여정 맵 (JW Financial 버전)

```
단계          시점              핵심 감정          대응
─────────────────────────────────────────────────────
탐색          상담 신청 전      "믿을 수 있나?"    콘텐츠 신뢰 구축
인정          상담 신청         "도움 필요해"      빠른 응답, 친절한 인테이크
확신          1~2차 미팅        "잘 선택했나?"     SPIN 질문, 가치 설계
활성화        포트폴리오 실행   "잘 한 거 맞지?"   D+0 환영 메시지
적응          D+1~90           "이게 맞는 건가?"  D+1/7/30 시퀀스 ← 가장 중요
성취          첫 성과 경험      "역시 잘 했어"     수익/혜택 가시화
채택          D+90 이후         "이 사람 믿어"     관리미팅, 추가 설계
옹호          장기 고객         "소개해줄게"       소개 요청 시퀀스
```

### 이탈 위험 신호 감지

```
🔴 즉시 연락 필요:
  - 메시지 2회 이상 무응답
  - 납입 이상 징후
  - 시장 급락 시 연락 없음

🟡 주의 관찰:
  - D+7/30/90 체크인 미발송
  - 3개월 이상 접촉 없음
  - 소개 의향 없는 장기 고객

🟢 정상:
  - 정기 체크인 응답
  - 콘텐츠 반응 있음
  - 추가 상담 요청
```

## FRAMEWORK 3 — NPS (Fred Reichheld)

**핵심 원칙:** "추천할 의향이 있는 고객이 진짜 충성 고객이다"

### 고객 분류 (JW Financial 3티어)

```
🌟 Promoters (추천자) — NPS 9~10점
  특징: 소개 의향 있음, 추가 계약 의향, 콘텐츠 공유
  관리: 소개 요청 시퀀스, VIP 케어
  목표: 이 그룹을 늘리는 것이 최우선

😐 Passives (중립) — NPS 7~8점
  특징: 만족하지만 소개는 안 함, 이탈 가능성 있음
  관리: D+90 관리미팅으로 Promoter로 전환 시도
  목표: 불만 요소 파악 → 제거

😞 Detractors (비추천자) — NPS 0~6점
  특징: 불만 있음, 이탈 위험
  관리: 즉시 1:1 통화, 문제 파악, 해결
  목표: 이탈 방지 + 원인 제거 (다음 고객에 반영)
```

## Notion CRM 운영 가이드

### 필수 필드

```
고객 기본정보:
  이름 / 연락처 / 직장 / 나이 / 가족구성

여정 관리:
  현재 단계 (탐색/확신/활성화/적응/성취/채택/옹호)
  포트폴리오 실행일 (D+ 카운터 기준)
  다음 액션 + 예정일
  위험 신호 여부

포트폴리오:
  상품 구성 (종류/금액/역할)
  총 포트폴리오 금액

관계 기록:
  최근 접촉일
  주요 메모 (가족 이벤트, 관심사, 우려사항)
  NPS 추정 등급
```

### 주간 CRM 루틴

```
매주 월요일:
  □ 이번 주 D+1/7/30/90 도래 고객 확인
  □ 이탈 위험 신호 고객 확인
  □ 미발송 메시지 처리

매주 금요일:
  □ 이번 주 접촉 고객 노션 업데이트
  □ 다음 주 팔로업 일정 세팅
  □ Promoter 고객 소개 요청 검토
```

## 불만 고객 대응 SOP

```
1. 즉시 응답 (24시간 이내)
   "불편을 드려서 죄송합니다. 바로 확인해드릴게요"

2. 사실 파악 (판단 전에)
   무슨 일인지 / 언제부터인지 / 원하는 것이 무엇인지

3. 해결책 제시
   가능한 것: 즉시 실행
   불가능한 것: 솔직하게 + 대안 제시

4. 사후 확인
   해결 후 3일 내 재연락 → "이제 괜찮으신가요?"

5. 재발 방지
   원인 파악 → SOP 업데이트
```
