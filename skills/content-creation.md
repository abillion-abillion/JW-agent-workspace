## 브랜드별 톤 & 매너

### JW Financial (고객 대상)
- **톤**: 친근하되 전문적, 형/언니처럼 솔직한 조언
- **구조**: 비유 먼저 → 원리 설명 → 개인 적용 → CTA
- **금지**: 전문 용어 남발, 수동적 표현, CTA 없는 마무리
- **CTA 예시**: "지금 무료 진단 받아보세요 → jwfinancial.co.kr/diagnosis.html"

### 핀사이트랩스 (리서치/매크로)
- **톤**: 분석적, 인과관계 중심, 기관 투자자 시각
- **구조**: 현상 → 원인 → 파급효과 → 개인 포트폴리오 시사점
- **소스**: 손에 잡히는 경제, 삼프로TV, 메르 블로그, 오건영, 한국경제

## 콘텐츠 유형별 형식

### 카드뉴스 (네이버/카카오)
```
[카드 1] 훅 (질문형 or 공감형)
[카드 2-4] 핵심 내용 (1카드 = 1메시지)
[카드 5] 요약
[카드 6] CTA
```

### 네이버 블로그 포스트
- 제목: 검색 키워드 포함, 숫자/혜택 명시
- 도입부: 독자 상황 공감 (2-3줄)
- 본문: 소제목 활용, 300자 내외 단락
- 마무리: 요약 + 상담 CTA
