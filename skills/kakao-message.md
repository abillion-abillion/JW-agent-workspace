# 스킬: 카카오 메시지 초안 (kakao-message)

> 고객 발송용 카카오 메시지 작성 시 참조. 발송 전 반드시 진우님 검토 필요.

---

## 메시지 유형별 템플릿

### 1. 신규 상담 후 팔로업
```
안녕하세요 [이름]님, 남진우 자산관리사입니다 😊
오늘 [주제] 관련 상담 감사했습니다.

말씀드린 내용 요약해서 보내드릴게요.
[핵심 내용 1-2줄]

궁금하신 점 언제든 편하게 연락 주세요!
```

### 2. 이벤트/혜택 안내
```
[이름]님, 좋은 소식 있어서 먼저 연락드려요 📌

[혜택 내용 - 1줄로]
→ 조건: [조건]
→ 기간: [기간]

관심 있으시면 말씀해 주세요!
남진우 드림
```

### 3. 정기 콘텐츠 공유
```
[이름]님 안녕하세요!
이번 주 꼭 알아두시면 좋을 정보 하나 공유드려요 👇

[핵심 메시지 1-2줄]

자세한 내용은 → [링크]
```

---

## 작성 원칙

1. **3줄 이내 핵심** - 모바일에서 스크롤 없이 읽히게
2. **이름 호명** - 첫 줄에 반드시 이름 포함
3. **이모지 절제** - 1-2개 최대, 과용 금지
4. **CTA 1개만** - 링크나 행동 유도는 하나로 집중
5. **발송 전 확인** - 결제/계약 관련 내용은 반드시 진우님 검토

---

## Solapi 연동 정보

- 채널: KakaoTalk 비즈채널 (Solapi)
- GAS 연동: Google Forms → GAS → Telegram 알림
- 자동 발송 가능 항목: 상담 확정 알림, 자료 링크 공유
