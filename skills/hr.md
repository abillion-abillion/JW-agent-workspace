## FRAMEWORK 1 — Netflix Culture (Patty McCord)

**핵심 원칙:** "뛰어난 동료가 최고의 복지다"

### 채용 기준 (JW Financial 맞춤)

```
필수 역량 (Must):
  □ 배우려는 의지 — 금융 지식은 교육 가능, 태도는 교육 불가
  □ 소통 능력 — 고객/진우님과 명확한 커뮤니케이션
  □ 실행력 — 말보다 행동, 마감 준수
  □ 디지털 친숙도 — 카카오/텔레그램/노션 기본 사용

우대 역량 (Nice to Have):
  □ 금융/경제 관심 (전공 불필요)
  □ 콘텐츠 제작 경험 (카드뉴스, 블로그 등)
  □ 이중언어 (한국어/중국어)

채용 거름망:
  "이 사람이 우리 팀에서 최고 성과자가 될 수 있는가?"
  → YES면 채용, 보통이면 패스
  (McCord 원칙: 평균 인재는 팀 수준을 낮춤)
```

### 역할별 채용 프로필

```
콘텐츠/운영 동료:
  - 카드뉴스 초안 작성 보조
  - SNS 채널 관리 (포스팅, 댓글)
  - 고객 팔로업 메시지 발송
  - 노션 CRM 업데이트

중국어권 담당 동료:
  - 샤오홍슈/위챗 채널 운영
  - 중국어 콘텐츠 번역/현지화
  - 중국어권 고객 초기 응대

금융 어시스턴트:
  - 포트폴리오 북 초안 보조
  - 시뮬레이션 데이터 정리
  - 상담 자료 준비
```

## FRAMEWORK 2 — Creativity Inc. (Ed Catmull)

**핵심 원칙:** "처음엔 누구나 못한다. 안전하게 시도할 수 있는 환경이 성장을 만든다"

### 온보딩 설계 (30/60/90일)

```
D+0 — 첫날
  □ JW Financial 비전/미션 공유
  □ AGENTS.md 전체 읽기
  □ 주요 툴 세팅 (텔레그램, 노션, 카카오채널)
  □ 첫 주 학습 계획 전달

D+1~30 — 이해 단계
  □ 콘텐츠 원칙 학습 (비유 먼저, CTA 1개, 판매 언어 금지)
  □ 기존 카드뉴스/블로그 50개 읽기
  □ 고객 여정 8단계 이해 (Coleman)
  □ 실제 업무 보조 시작 (초안 → 진우님 피드백 반복)

D+31~60 — 실행 단계
  □ 독립적으로 초안 작성 → 검토 → 개선 반복
  □ 카카오 메시지 시퀀스 직접 작성
  □ 노션 CRM 자체 관리
  □ 주간 1:1 피드백 미팅 (15분)

D+61~90 — 자립 단계
  □ 담당 고객군 배정
  □ 콘텐츠 채널 1개 독립 운영
  □ 90일 리뷰: 역할 확대 or 조정
```

### 교육 커리큘럼 (금융 입문 4주)

```
Week 1: JW Financial 이해
  - 우리 고객은 누구인가 (30대 맞벌이)
  - 주력 상품군 개요 (변액/IRP/ISA/보험)
  - 콘텐츠 원칙 10가지

Week 2: 고객 커뮤니케이션
  - 카카오 메시지 원칙 (kakao-message.md 숙지)
  - 판매 언어 vs 교육 언어 구분
  - 실습: 메시지 초안 5종 작성

Week 3: 콘텐츠 제작
  - 카드뉴스 구조 (7컷 표준)
  - 블로그 포스팅 구조
  - 실습: 카드뉴스 1개 완성

Week 4: 운영 도구
  - 노션 CRM 사용법
  - 텔레그램 봇 활용
  - 주간 루틴 세팅
```

## FRAMEWORK 3 — Peter Drucker (사람 중심 경영)

**핵심 원칙:** "강점을 활용하라. 약점 보완에 에너지 낭비하지 마라"

### 역할 설계 4단계

```
1. 목적: 이 역할이 존재하는 이유
2. 결과: 성과가 나오면 어떻게 보이는가
3. KPI: 측정 방법
4. 강점: 이 역할에 맞는 사람의 특성

예시 — 콘텐츠 담당:
  목적: 진우님이 콘텐츠에 쓰는 시간 50% 절감
  결과: 주 2개 카드뉴스, 월 4개 블로그 포스팅 독립 발행
  KPI: 발행 수, 반응률, 진우님 수정 횟수 감소 추이
  강점: 글쓰기, 정보 정리, 디자인 감각
```

### 주간 1:1 피드백 구조 (15분)

```
1. 잘 된 것 1가지 (구체적으로)
   "이번 카드뉴스 첫 문장이 좋았어요. 고객 상황 딱 집어서"

2. 개선할 것 1가지 (행동 중심)
   "이 부분은 판매 언어예요. 이렇게 바꿔보면 어떨까요?"
   → 판단 금지, 행동 피드백만

3. 다음 주 목표 1가지
   "다음 주는 D+30 메시지 3개 독립으로 작성해보세요"
```

## 팀 문화 원칙 (JW Financial)

```
1. 고객 먼저 — 모든 판단 기준: "고객에게 도움이 되는가"
2. 교육 문화 — 모르면 물어보는 게 당연, 알면 가르치는 게 당연
3. 문서화 습관 — 잘 된 것은 즉시 SOP로 (머릿속은 자산 아님)
4. 빠른 실행 — 완벽보다 작동하는 것 먼저
5. 솔직한 피드백 — 좋은 게 좋은 게 아님, 성장이 목표
```
