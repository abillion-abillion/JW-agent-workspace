# 스킬: 소개 요청 구조화 (skill-referral)

> 관리미팅마다 구조화된 소개 요청 스크립트 제공
> 현재: 비구조적 구두 요청 → 개선: 맥락 있는 스토리 + 명확한 타겟 + 소개카드

---

## 트리거
관리미팅 전날 진우님 Telegram으로 자동 알림:
"내일 [고객명] 관리미팅입니다. 소개 요청 스크립트 준비됐습니다."

또는 진우님: "○○ 소개 요청 스크립트 만들어줘"

---

## 스크립트 구조

### 파트 1. 나의 성장 스토리 (1~2분)
> 목적: 소개할 만한 사람이라는 신뢰 구축

```
"요즘 제가 ○○ 쪽으로 성장하고 있어서요.
[최근 변화/성과 1줄 - 진우님이 직접 채워야 함]
덕분에 요즘 더 도움드릴 수 있는 범위가 넓어졌어요."
```

예시:
- "최근에 핀사이트랩스라는 리서치 브랜드를 시작했어요"
- "IFP 자격증 준비 중이에요"
- "네이버 고객분들 위한 전용 포트폴리오 서비스를 만들었어요"

---

### 파트 2. 타겟 명확화
> 목적: "누구든 소개해줘"가 아니라 구체적 타겟 제시

```
"혹시 주변에 이런 분 계신가요?

- 30대 직장인이신데 재무 정리를 한 번도 못 하신 분
- 보험은 있는데 제대로 된 건지 모르시는 분
- IRP나 연금저축 계좌는 있는데 뭘 담아야 할지 모르시는 분

딱 한 분이라도 생각나시면 말씀해 주시면 감사하겠습니다 😊"
```

---

### 파트 3. 소개 방법 구체화
> 목적: 행동 장벽 최소화

```
"소개해주실 때 이렇게 해주시면 돼요.

그냥 제 카카오톡 연락처 전달해 주시거나,
이 소개카드를 카톡으로 보내주시면 됩니다.

→ [QR 소개카드 링크 또는 이미지]

부담 없이 한 번 만나보시라고만 해주셔도 충분해요!"
```

## 관리미팅 전날 알림 포맷

```
📅 내일 관리미팅 알림

고객: [고객명]
시간: [시간]
형태: 대면 / 줌

━━━━━━━━━━━━━━
소개 요청 포인트:
• 최근 성과: [진우님이 업데이트]
• 추천 타겟: 30대 직장인, 재무 정리 필요한 분
• 소개 방법: 소개카드 전달

소개카드 링크: jwfinancial.co.kr/consult.html
━━━━━━━━━━━━━━
```
