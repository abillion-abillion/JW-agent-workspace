## 언제 이 스킬을 쓰는가

대장은 아래 키워드 포함 시 이 스킬을 부장에게 전달한다:
- `카드뉴스`, `인스타`, `카드`, `이미지 만들어`, `SNS 이미지`
- `3장`, `슬라이드`, `카드뉴스 만들어`

---

## 부장 작업 지시 템플릿

대장이 부장에게 이 스킬 적용 시 아래 형식으로 지시:

```
[카드뉴스 생성 지시]
주제: {주제}
채널: {인스타그램 / 카카오채널 / 블로그 썸네일}
톤: {JW Financial 친근+전문 / 핀사이트랩스 분석적}
구성:
  1장 — 훅 (고객 공감 포인트, 체크리스트 형식)
  2장 — 핵심 내용 (3회 프로세스 or 주제 설명)
  3장 — 준비사항 + CTA (jwfinancial.co.kr)
디자인 원칙:
  - 배경: 다크 네이비 #07192e
  - 액센트: 골드 #c9a96e
  - 폰트: Noto Serif KR (헤드라인) + Noto Sans KR (본문)
  - 크기: 1080×1080px
  - 카드번호: 01/03, 02/03, 03/03 표시
출력: HTML 3장 (---CARD_SPLIT--- 구분자 사용)
```

---

## 부장 HTML 생성 규칙

### 공통 CSS 변수 (3장 모두 동일하게 적용)
```css
배경: linear-gradient(155deg, #0d2340 0%, #07192e 45%, #040f1c 100%)
골드: #c9a96e
텍스트: rgba(255,255,255,0.82)
카드크기: 1080px × 1080px
패딩: 72px 80px 64px
구분자: ---CARD_SPLIT---
```

### 1장 구성 (훅 카드)
- 상단: `PERSONAL FINANCIAL CONSULTING` 라벨 + `01 / 03`
- 헤드라인: Noto Serif KR Bold, 48px
- 체크리스트: 6개 이내, 골드 체크아이콘, 반투명 박스
- 하단: 골드 CTA 문구

### 2장 구성 (내용 카드)
- 상단: `HOW IT WORKS` 또는 주제 라벨 + `02 / 03`
- 번호 배지: 골드 원형, 연결선으로 스텝 표현
- 각 스텝: 제목(Noto Serif) + 설명(Noto Sans, 회색)
- 하단: 인포박스 (골드 테두리)

### 3장 구성 (CTA 카드)
- 상단: `BEFORE YOUR CONSULTATION` 또는 요약 라벨 + `03 / 03`
- 준비사항: 3개 번호 배지 + 좌측 골드 라인 박스
- 하단 2열: 가격박스(골드) + CTA박스(jwfinancial.co.kr)

## 대장 검토 기준

부장이 HTML을 생성하면 대장은 아래를 확인:
- [ ] 3장 모두 `---CARD_SPLIT---` 구분자로 분리됐는가
- [ ] 각 카드에 `XX / 03` 번호가 있는가
- [ ] CTA에 `jwfinancial.co.kr` 포함됐는가
- [ ] 1080×1080px 크기 설정됐는가
- [ ] 다크 네이비 + 골드 디자인 준수했는가
- [ ] HTML 코드 외 불필요한 텍스트 없는가

이상 없으면: `✅ 대장 검토 완료 — PNG 변환 시작`
이상 있으면: 구체적으로 재지시
