## 언제 이 스킬을 쓰는가

대장은 아래 키워드 포함 시 이 스킬을 부장에게 전달한다:
- `카카오`, `카톡`, `문자`, `메시지 보내줘`
- `2차완료`, `계약완료`, `온보딩`
- `이탈방지`, `팔로업`, `체크인`
- `고객에게 연락`, `단체발송`, `시퀀스`
- `감사 메시지`, `안부`, `리뷰 요청`, `소개 요청`

## 메시지 발송 원칙 (절대 규칙)

```
1. 이름 호명 필수 — "[이름]님" 으로 시작
2. 3줄~5줄 이내 — 카카오는 미리보기가 전부
3. CTA 1개만 — 링크 2개, 행동 2개 금지
4. 이모지 최대 2개 — 남발하면 신뢰도 하락
5. 금지 단어 — 가입, 계약, 판매, 납입 관련 표현 일체 금지
6. 발송 전 진우님 확인 — ✅ 검토 후 발송 표기 필수
7. 개인화 — 고객 실제 포트폴리오/상황/날짜 반영, 복붙 티 나면 안 됨
```

## FRAMEWORK — Never Lose a Customer Again (Joey Coleman)

**핵심 원칙:** "함께한 후 100일이 평생 고객을 만드는 결정적 시간이다"

**JW Financial 고객 여정 매핑:**
```
상담 신청         → 탐색/인정 단계
1차 미팅          → 확신 불안 시작
2차 미팅          → 확신 불안 지속 + 3차 미팅 약속 완료
포트폴리오 실행   → 활성화
실행 후 D+1~90   → 적응 ← 이탈 최다 구간
첫 수익/혜택      → 성취
추가 관리/재설계  → 채택
소개/레퍼럴       → 옹호
```

## 메시지 유형별 템플릿

### A. 단발 메시지 (상황별)

#### A-1. 신규 상담 후 팔로업
```
[이름]님, 남진우 자산관리사입니다 😊
오늘 [주제] 관련 상담 감사했습니다.

말씀드린 내용 요약해서 보내드릴게요.
[핵심 내용 1~2줄]

궁금하신 점 언제든 편하게 연락 주세요!

✅ 검토 후 발송해주세요
⚠️ [이름], [주제], [핵심내용] 확인필요
```

#### A-2. 정기 콘텐츠 공유
```
[이름]님 안녕하세요!
이번 주 알아두시면 좋을 정보 공유드려요 👇

[핵심 메시지 1~2줄]

자세한 내용은 → [링크]

✅ 검토 후 발송해주세요
⚠️ [이름], [핵심내용], [링크] 확인필요
```

#### A-3. 미팅 전날 리마인더
```
[이름]님, 내일 [시간] 미팅 확인 차 연락드려요!

장소: [장소]
따로 준비하실 건 없으세요, 편하게 오시면 됩니다 😊

✅ 검토 후 발송해주세요
⚠️ [이름], [시간], [장소] 확인필요
```

#### A-4. 자료 전달
```
[이름]님, 지난번에 말씀드린 자료 보내드려요!

→ [링크 or 파일]

보시고 궁금한 거 있으면 편하게 연락 주세요.

✅ 검토 후 발송해주세요
⚠️ [이름], [링크/파일] 확인필요
```

#### A-5. 시장 급변 시 선제 연락
```
[이름]님, 오늘 시장 변동이 있어서 먼저 연락드려요.

[이름]님 포트폴리오에는 이런 영향이 있어요:
[한 줄 요약]

걱정하실 필요 없는 이유도 설명드릴 수 있어요.
잠깐 통화 가능하세요?

✅ 검토 후 발송해주세요
⚠️ [이름], [영향 내용] 확인필요
```

#### A-6. 세제혜택 시즌 알림 (10~12월)
```
[이름]님, 연말정산 시즌이 다가왔어요!

IRP/연금저축 한도 확인하셨나요?
지금 채우시면 [○○]만원까지 돌려받으실 수 있어요.

한도 확인해드릴까요? 말씀 주세요 😊

✅ 검토 후 발송해주세요
⚠️ [이름], [금액] 확인필요
```

#### A-7. 생일 축하
```
[이름]님, 생일 축하드려요! 🎂

항상 건강하시고 좋은 일만 가득하시길 바랍니다.
올해도 재무 목표 함께 챙겨드릴게요!

✅ 검토 후 발송해주세요
⚠️ [이름] 확인필요
```

#### A-8. 월별 시황 공유 (전체 고객)
```
[이름]님, [월] 시장 흐름 요약드려요.

[핀사이트랩스 이번 달 핵심 1줄]

자세한 내용은 여기서 보실 수 있어요:
→ [링크]

궁금한 점 있으면 편하게 말씀 주세요!

✅ 검토 후 발송해주세요
⚠️ [이름], [월], [핵심내용], [링크] 확인필요
```

### B. 이탈방지 시퀀스 (2차완료 후 3차 미팅 전)

> 트리거: `2차완료 [고객명]`
> 목적: 2차→3차 사이 이탈 방지 (포트폴리오 이해도 제고, 확신 불안 해소)
> ※ 3차 미팅은 2차에서 이미 약속 완료 — 일정 재확인 불필요

#### D+1 — 포트폴리오 상기 + 상품별 역할·효율성 설명
```
[이름]님, 어제 미팅 감사했습니다 😊

오늘은 [이름]님 포트폴리오를 다시 한 번 짚어드릴게요.

[상품 A]는 [목적]을 위해,
[상품 B]는 [목적]을 위해 담았어요.
각각의 역할이 다르기 때문에 함께 쓸 때 효율이 높아지는 구조입니다.

궁금한 점 생기시면 편하게 연락 주세요!

✅ 검토 후 발송해주세요
⚠️ [이름], [상품명 A/B], [각 목적] 확인필요
```
> 부장 주의: 해당 고객 실제 포트폴리오 구성 반영 필수 / 상품 나열이 아닌 역할·시너지 중심 설명

#### D+2 — 비식별 사례 공유
```
[이름]님, 안녕하세요!

비슷한 상황의 분 사례 공유드려요.
(개인정보는 모두 제외했습니다)

[○○]만원 포트폴리오 실행 → [○○]년 후 결과
→ [사례 요약 or 링크]

[이름]님 상황에 맞는 시뮬레이션도 가능해요.
궁금하신 거 있으면 말씀 주세요!

✅ 검토 후 발송해주세요
⚠️ [이름], [포트폴리오 금액], [기간], [결과 수치] 확인필요
```

#### D+3 — 기대 혜택 가시화
```
[이름]님!

지금 실행하신 포트폴리오를 꾸준히 함께 관리하면
[○○]년 후 이런 그림이 그려집니다.

• 세금 절감: 연 [○○]만원 환급 효과
• 자산 누적: [○○]세 시점 예상 [○○]억
• 보장 안전망: [위험 상황] 발생 시 [○○] 커버

숫자로 보니 조금 더 선명하게 느껴지시죠?
다음 미팅에서 더 자세히 이야기 나눠요 😊

✅ 검토 후 발송해주세요
⚠️ [이름], [기간], [세금절감액], [예상자산], [나이], [보장내용] 확인필요
```
> 부장 주의: 고객 실제 포트폴리오 기준 수치 구체화 / 3차 미팅 일정 재언급 불필요

### C. 온보딩 시퀀스 (포트폴리오 실행 완료 후 D+0~90)

> 트리거: `계약완료 [고객명]`
> 목적: 실행 후 불안 해소 → 신뢰 구축 → 장기 고객화

#### D+0 — 당일 환영
```
[이름]님, 오늘 함께하게 되어 정말 기쁩니다! 🎉

앞으로 꾸준히 챙겨드릴게요.
궁금한 거 생기면 언제든 편하게 연락 주세요.

첫 번째 체크인은 일주일 후에 드릴게요.
오늘 수고 많으셨습니다 😊

✅ 검토 후 발송해주세요
⚠️ [이름] 확인필요
```

#### D+7 — 첫 체크인 (포트폴리오 전반 진행 안내)
```
[이름]님, 함께한 지 일주일 됐네요!

포트폴리오 전반이 이렇게 진행됩니다:

• [보장 상품]: [○○]일 자동이체 출금 예정
• [증권 투자 상품]: 매월 [○○]일 자동매수 적립식 세팅 완료
  → 매달 자동으로 [○○] 자산을 쌓아가는 구조예요

각 항목이 자동으로 굴러가도록 세팅되어 있으니
따로 신경 쓰실 건 없어요.
혹시 확인하실 게 있으면 말씀 주세요!

✅ 검토 후 발송해주세요
⚠️ [이름], [날짜], [상품명], [자동이체/자동매수 일정] 확인필요
```

#### D+30 — 한 달 리뷰
```
[이름]님, 함께하게 된 지 벌써 한 달 됐네요!

처음 미팅하셨을 때와 비교해
경제 환경이 이렇게 달라졌어요:

• [주요 경제/정책 변화 1]
• [주요 이슈 2]

이런 흐름 속에서도 [이름]님 자산은
계획대로 잘 모이고 있습니다 😊

다음 달도 함께 잘 챙겨드릴게요!

✅ 검토 후 발송해주세요
⚠️ [이름], [경제/정책 이슈 — 발송 시점 기준 최신 내용으로 업데이트] 확인필요
```

#### D+90 — 1차 관리미팅 제안
```
[이름]님, 함께 관리받게 되신 지 3개월이 되었습니다.

내부·외부적으로 변경된 상황이 있을 테니
한 번 이야기 나누고 싶어요.

1차 관리미팅 일정 잡아도 될까요?
약 1시간 정도 예상됩니다.
→ [예약 링크]

✅ 검토 후 발송해주세요
⚠️ [이름], [링크] 확인필요
```

### D. 소개 요청 시퀀스 (Advocate 단계)

> 트리거: `소개 요청 - [고객명]`
> 타이밍: D+90 이후, 관계 충분히 쌓인 고객
> 원칙: Give → Take 순서. 가치 있는 것을 먼저 드리고 자연스럽게 연결

#### 먼저 드릴 것 선택 (Give 옵션)

| 옵션 | 내용 | 적합한 경우 |
|------|------|------------|
| ① 맞춤 재무진단 보고서 | 현금흐름 + 순자산 + 갭 분석 + 상품 조합 심화 자료 | 재무 상황 복잡한 지인 소개 시 |
| ② 핀사이트랩스 리포트 1개월 | 매크로 분석 + 투자 시사점 심화 버전 | 투자 관심 높은 지인 소개 시 |
| ③ 재무 체크리스트 키트 | "30대 맞벌이가 지금 확인해야 할 10가지" + 1:1 리뷰 세션 | 금융 입문 단계 지인 소개 시 |

> 부장은 고객 상황에 맞는 옵션 1개 선택해서 메시지에 반영

#### 소개 요청 메시지
```
[이름]님, 요즘 어떻게 지내세요?

한 가지 먼저 드리고 싶은 게 있어요.

[이름]님처럼 꼼꼼하게 관리받고 계신 분들께
[① 맞춤 재무진단 보고서 / ② 핀사이트랩스 리포트 1개월 / ③ 재무 체크리스트 키트]를
따로 준비해드리고 있어요.

주변에 재무 고민 있는 분 계시면
편하게 소개해 주세요 😊
소개해주신 분께도 똑같이 정성껏 챙겨드릴게요.

부담 없이 말씀해 주세요!

✅ 검토 후 발송해주세요
⚠️ [이름], [증정 콘텐츠 종류 선택] 확인필요
```

## 톤 & 스타일 가이드

```
✅ 이런 표현으로:
  "함께하게 된 지" (가입/계약 대신)
  "공유드려요" (안내드립니다 대신)
  "포트폴리오 실행" (가입/계약 대신)
  "편하게 연락 주세요" (문의 주시기 바랍니다 대신)

❌ 절대 금지 표현:
  가입, 계약, 판매, 납입
  "귀하께서는..." (공문 느낌)
  "본 상품은..." (브로셔 느낌)
  "특별 혜택을 드립니다" (영업 느낌)
  "지금 바로 신청하세요" (압박 느낌)
```
