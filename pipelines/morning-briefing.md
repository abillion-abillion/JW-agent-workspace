# 파이프라인: 모닝 브리핑

> 상태: ✅ 운영중

---

## 개요

매일 아침 주요 경제/금융 뉴스를 수집하여 Telegram으로 요약 발송합니다.

---

## 플로우

```
RSS 피드 수집 (경제/금융 주요 소스)
  ↓
Claude API 요약 처리
  ↓
Telegram Bot 발송 (진우님 개인 채널)
```

---

## 소스 목록

- 손에 잡히는 경제
- 삼프로TV
- 메르 블로그
- 오건영 Facebook 칼럼
- 한국경제 (한경)

---

## 출력 형식

```
📊 [날짜] JW 모닝 브리핑

🔑 오늘의 핵심
- [요약 1]
- [요약 2]
- [요약 3]

💼 고객 상담 활용 포인트
- [활용 포인트]

📌 오늘 챙길 것
- [할 일]
```

---

## 기술 스택

- 언어: Python
- 스케줄러: APScheduler (or cron)
- API: Claude API (claude-sonnet)
- 발송: Telegram Bot API
