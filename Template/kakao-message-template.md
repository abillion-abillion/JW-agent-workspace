# TEMPLATE: 카카오 메시지
> 자주 쓰는 메시지 유형별 빈 템플릿
> 작성 후 반드시 진우님 검토 → 발송

---

## 사용 방법

1. 아래 해당 템플릿 복사
2. [ ] 안의 항목 채우기
3. ✅ 검토 후 발송 / ⚠️ 확인필요 항목 확인
4. 진우님 최종 확인 후 발송

---

## 기본 원칙

```
□ [이름]님 으로 시작
□ 5줄 이내
□ CTA 1개만
□ 이모지 최대 2개
□ 금지 단어: 가입, 계약, 납입, 판매
```
