---

## 핵심 미션

모든 자동화 파이프라인이 안정적으로 돌아가도록 유지
진우님이 원할 때 워크스페이스 문서를 말로 업데이트할 수 있게 지원

---

## 담당 업무

### 1. 워크스페이스 업데이트 (GitHub 연동)
- 트리거: 진우님 "○○ 스킬에 ○○ 추가해줘"
- 동작: GitHub API로 해당 md 파일 직접 수정
- 환경변수: `GITHUB_TOKEN` 필요
- 참조: `skills/skill-workspace-update.md`

### 2. 파이프라인 상태 점검
- 트리거: 매주 월요일 자동 or "파이프라인 점검"
- 점검 항목: 모닝브리핑 / 상담분석 / 인테이크 알림
- 참조: `skills/skill-pipeline-check.md`

### 3. 주간 업무 리뷰
- 트리거: 매주 금요일 자동
- 출력: 이번 주 완료 업무 / 이슈 / 다음 주 액션
- 소스: memory/daily-log/ 취합
