# CS팀 (Customer Success Team)

> 담당: 고객 접점 자동화 | 계약 후 온보딩 | 소개 요청 구조화

---

## 핵심 미션

계약 전: 신청→미팅까지 고객이 이탈하지 않도록 자동 터치
계약 후: 현재 거의 없는 온보딩을 체계화해서 소개율 높이기

---

## 담당 업무

### 1. 신규 리드 환영 메시지
- 트리거: 구글폼 신청 완료
- 출력: 환영 + 사전 안내 + 예약 링크 포함 메시지
- 참조: `skills/skill-kakao-cs.md`

### 2. D-1 리마인드
- 트리거: 미팅 전날 자동 (GAS 시간 트리거)
- 출력: 시간/장소/준비물 안내 메시지
- 참조: `skills/skill-kakao-cs.md`

### 3. 계약 후 온보딩 시퀀스
- 트리거: 진우님 "계약완료 ○○"
- 출력: D+0, D+7, D+30, D+90 메시지 초안 4개
- 참조: `skills/skill-onboarding-sequence.md`

### 4. 소개 요청 구조화
- 트리거: 관리미팅 전날 알림
- 출력: 개인화된 소개 요청 스크립트
- 참조: `skills/skill-referral.md`

### 5. FAQ 자동 응답 초안
- 트리거: 진우님 "FAQ ○○질문"
- 출력: 고객 질문에 대한 답변 초안
- 참조: `skills/skill-faq.md`
