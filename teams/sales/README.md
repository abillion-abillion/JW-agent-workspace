# 영업팀 (Sales Team)

> 담당: 상담 프로세스 지원 | 포트폴리오 북 자동화 | 이탈 방지

---

## 핵심 미션

3차 미팅까지 고객을 끊기지 않게 이어주는 것.
진우님이 상담에만 집중할 수 있도록 준비 작업을 대신 처리.

---

## 담당 업무

### 1. 포트폴리오 북 초안 생성
- 트리거: 진우님 "○○ 포트폴리오 북 만들어줘"
- 입력: 노션 고객 DB (상담일지 요약)
- 출력: 섹션별 초안 텍스트 (재무현황 / SWOT / 목표 / 포트폴리오 / 상품설명)
- 참조: `skills/skill-portfolio-book.md`

### 2. 2차→3차 이탈 방지 시퀀스 초안
- 트리거: 진우님 "2차완료 ○○"
- 출력: D+1, D+2, D+3 발송용 메시지 초안 3개
- 참조: `skills/skill-dropout-prevention.md`

### 3. 미팅별 준비자료 생성
- 1차 준비: 고객 사전 정보 기반 체크리스트
- 2차 준비: 상품 비교표 초안 (은행/증권/보험)
- 3차 준비: 포트폴리오 북 최종 검토 포인트
- 참조: `skills/skill-meeting-prep.md`

### 4. ETF 당일 포트폴리오 정리
- 트리거: 진우님 "오늘 ETF 포트폴리오 정리해줘"
- 출력: 프린트용 ETF 구성표 텍스트 초안
- 참조: `skills/skill-etf-portfolio.md`

## 상담 프로세스 구조

```
1차 미팅 (재무 진단)
  현금흐름 / 재무상태표 / 보험 효율성
  재무목표 설정 / 투자원칙 안내
  → 상담일지 녹음 → 텍스트 변환 → 노션 저장
  ↓ 3~4일
2차 미팅 (상품 안내)
  목표별 금융상품 설명 (은행/증권/보험)
  금액 조율 / 부담감 제거
  ⚠️ 최대 이탈 구간
  → D+1,2,3 이탈방지 콘텐츠 자동 발송
  ↓ 3~4일
3차 미팅 (계약)
  포트폴리오 북 전달
  금융상품 계약 (ISA, IRP, 연금저축, 보험 등)
  소개 요청
```
