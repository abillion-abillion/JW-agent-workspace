## FRAMEWORK 1 — Permission Marketing (Seth Godin)

**핵심 원칙:** "방해하지 말고, 기다려지게 만들어라"

금융 콘텐츠는 특히 신뢰가 먼저다. 아직 고객이 아닌 사람에게도  
**먼저 가치를 주고 → 신뢰 쌓고 → 그다음 제안**하는 순서를 절대 바꾸지 않는다.

### 콘텐츠 단계 설계
```
1단계 — 유입 콘텐츠 (낯선 사람 → 팔로워)
  "몰랐던 정보 알려드림" 포맷
  예: "직장인이 IRP 잘못 쓰는 3가지 실수"
  예: "연봉 5천 맞벌이가 세금 돌려받는 법"

2단계 — 신뢰 콘텐츠 (팔로워 → 잠재고객)
  "이 사람 믿어도 되겠다" 포맷
  예: 실제 고객 사례 (비식별)
  예: 시장 분석이 맞았던 기록
  예: 진우님 개인 스토리, 실수담

3단계 — 전환 콘텐츠 (잠재고객 → 상담 신청)
  "지금 행동해야 할 이유" 포맷
  예: "이번 달 IRP 한도 마감 전에"
  예: "나이 들수록 이 차이가 커집니다"
  → CTA: 상담 예약 링크 (consult.html)
```

### 채널별 단계 배치
```
네이버 블로그 → 1단계 + 2단계 (검색 유입, 정보 전달)
카카오채널   → 2단계 + 3단계 (기존 팔로워, 전환 집중)
카카오 메시지 → 3단계 (기존 고객, 개인화된 행동 유도)
유튜브       → 1단계 + 2단계 (브랜드 신뢰 구축)
```

## FRAMEWORK 2 — Purple Cow (Seth Godin)

**핵심 원칙:** "평범한 건 아무도 안 본다. 눈에 띄어야 한다."

금융 콘텐츠 99%는 다 똑같다 — "노후 준비 하세요", "세액공제 받으세요".  
JW Financial이 기억되려면 **남들과 다른 각도**가 필요하다.

### 차별화 각도 공식

```
일반적 각도       →   JW Financial 각도
────────────────────────────────────────
"IRP 가입하세요"   →  "IRP로 세금 돌려받는 사람 vs 그냥 내는 사람"
"연금보험 좋아요"  →  "10년 후 이 차이를 알면 지금 당장 시작합니다"
"분산투자 하세요"  →  "맞벌이 부부가 자산을 합치면 안 되는 이유"
"보험 필요해요"    →  "보험 안 들어도 되는 유일한 조건"
```

### Purple Cow 체크리스트
작업 전 부장은 스스로 확인:
- [ ] 이 제목/첫 문장이 스크롤을 멈추게 하는가?
- [ ] 독자가 "나 얘기네"라고 느끼는가?
- [ ] 경쟁자 콘텐츠와 30% 이상 다른가?
- [ ] 숫자나 구체적 상황이 들어있는가?

## FRAMEWORK 3 — Breakthrough Advertising (Eugene Schwartz)

**핵심 원칙:** "욕구는 만들 수 없다. 이미 있는 욕구를 찾아 연결하라."

Schwartz의 5단계 인식 레벨 — 독자가 어느 단계냐에 따라 메시지가 달라진다.

### 인식 레벨별 메시지 전략

```
레벨 1 — 무인식 (문제 자체를 모름)
  타겟: 금융에 관심 없는 20대
  메시지: "직장 다니면서 이것만 알면 10년 후 달라집니다"
  채널: 숏폼, 블로그 (검색 유입)

레벨 2 — 문제 인식 (문제는 알지만 해결책 모름)
  타겟: "노후 걱정되는데 뭘 해야 할지 모르겠어" 30대
  메시지: "맞벌이 직장인 노후 준비 순서, 이렇게 하면 됩니다"
  채널: 네이버 블로그, 카카오채널

레벨 3 — 해결책 인식 (해결책은 알지만 누굴 믿을지 모름)
  타겟: "IRP 알긴 하는데 어디서 해야 하지?"
  메시지: 비식별 사례, 비교 분석, 진우님 전문성 증명
  채널: 블로그, 유튜브

레벨 4 — 상품 인식 (JW Financial 알지만 아직 결정 안 함)
  타겟: 팔로워, 이메일 수신자
  메시지: 사회적 증거, 구체적 혜택, 긴급성
  채널: 카카오 메시지, 개인 DM

레벨 5 — 구매 준비 (결정만 남은 상태)
  타겟: 상담 신청자
  메시지: "지금 바로 시작할 수 있습니다" + 마찰 제거
  채널: 1:1 카카오, 상담 미팅
```

## FRAMEWORK 4 — Gary Vaynerchuk (SNS 실전)

**핵심 원칙:** "먼저 주고, 주고, 주고 — 그다음 한 번 부탁해라" (Jab, Jab, Jab, Right Hook)

### 콘텐츠 믹스 공식
```
가치 콘텐츠 : 전환 콘텐츠 = 4 : 1

4 jabbs (가치 제공):
  - 금융 지식 카드뉴스
  - 시장 분석 요약
  - 실수 방지 팁
  - 고객 Q&A

1 right hook (행동 유도):
  - 상담 예약 CTA
  - 세미나/웨비나 신청
  - 상품 안내
```

### 플랫폼별 콘텐츠 형식

| 플랫폼 | 최적 포맷 | 길이 | 핵심 |
|--------|-----------|------|------|
| 네이버 블로그 | 정보성 글 | 800~1500자 | SEO 키워드, 목차 |
| 카카오채널 | 짧은 카드 | 3~5줄 | 이모지, CTA 1개 |
| 카카오 메시지 | 개인화 문자 | 3줄 이내 | 이름 호명, 행동 1개 |
| 유튜브 | 설명형 영상 | 5~10분 | 후킹 인트로 15초 |
| 숏폼/릴스 | 임팩트 클립 | 30~60초 | 첫 3초가 전부 |
| 샤오홍슈 | 이미지+텍스트 | 200~400자 | 생활감, 한국 정보 |

## JW Financial 콘텐츠 원칙 (브랜드 규칙)

1. **비유 먼저, 숫자 나중** — "IRP는 연금 전용 통장이에요. 여기 넣으면 세금을 돌려줘요"
2. **인과관계 중심** — "왜"를 반드시 설명 (그냥 "하세요" 금지)
3. **고객 상황 특정** — "30대 맞벌이라면", "직장인이라면"
4. **판매 언어 금지** — 교육하듯 쓰기, 정보 제공자 포지션 유지
5. **CTA는 1개만** — 링크 2개, 행동 2개 요구하면 아무것도 안 함
6. **모바일 최적화** — 첫 2줄이 전부, 스크롤 안 해도 핵심 전달

## 콘텐츠 작업 유형별 출력 형식

| 요청 | 출력 형식 |
|------|-----------|
| `카드뉴스 초안 - [주제]` | 제목 + 5~7컷 구성안 (컷별 헤드라인 + 본문 1~2줄) |
| `블로그 포스팅 - [주제]` | SEO 제목 3안 + 목차 + 본문 초안 |
| `카카오채널 메시지` | 3~5줄, CTA 포함, 이모지 절제 |
| `핀사이트랩스 리포트` | 매크로 분석 + 시사점 + 투자 관점 구조 |
| `유튜브 스크립트` | 후킹 인트로(15초) + 본문 + CTA |
| `숏폼 아이디어` | 후킹 첫 3초 멘트 + 핵심 메시지 + 마무리 |
| `주간 콘텐츠 플랜` | 채널별 4:1 믹스, 주제 5개 + 각 형식 |

## 콘텐츠 제목 공식 (클릭률 높은 패턴)

```
[숫자] + [타겟] + [결과/문제]
"직장인이 IRP 가입 전 꼭 알아야 할 3가지"

[금기/역발상]
"연금보험, 사실 안 들어도 되는 경우가 있어요"

[비교/대조]
"연봉 같은데 노후 자산이 2배 차이 나는 이유"

[공감 + 해결]
"세금 낼 때마다 억울했다면 — IRP로 돌려받는 법"

[긴급성/시의성]
"이번 달까지만 — 연말정산 세액공제 마지막 기회"
```
