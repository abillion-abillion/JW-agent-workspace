# 스킬: 2차→3차 이탈 방지 시퀀스 (skill-dropout-prevention)

> 2차 미팅 후 3일간 발송할 메시지 초안 자동 생성
> 목적: 보험 생소함 해소 + 3차 미팅 거부감 제거

---

## 트리거
진우님: "2차완료 ○○" 또는 "○○ 이탈방지 시퀀스"

---

## 배경 (왜 이게 필요한가)

2차 미팅에서 보험 상품이 포트폴리오에 포함됨.
고객 입장에서:
- 보험 = 뭔가 팔리는 느낌 → 불편함
- 3차 = 결정해야 하는 자리 → 회피

→ 아무 연락 없이 3차 미팅을 잡으면 이탈 확률 ↑
→ 3일간 교육 + 프레이밍 콘텐츠로 선제 대응

---

## D+1 메시지 (2차 미팅 다음날)

**목적:** 보험이 포트폴리오에 들어가는 이유 교육

```
[고객명]님, 어제 미팅 수고 많으셨어요 😊

오늘은 어제 말씀드린 보험 관련해서
짧게 하나만 공유드리고 싶어서요.

많은 분들이 "왜 보험이 재무 포트폴리오에 들어가나요?"
라고 궁금해하시는데요.

재무 포트폴리오에서 보험의 역할은
'수익을 내는 것'이 아니라
'이미 만들어 놓은 자산을 지키는 것'입니다.

예를 들어, 10년 동안 1억을 모았는데
한 번의 입원비/수술비로 수천만원이 나간다면
그 자산이 흔들리게 되죠.

[고객명]님 포트폴리오에서 보험이 맡는 역할은
바로 이 '방어선' 역할입니다.

다음 미팅에서 더 자세히 설명드릴게요!

남진우 드림
✅ 검토 후 발송해주세요
```

---

## D+2 메시지 (2차 미팅 이틀 후)

**목적:** 비식별 사례로 의사결정 앵커링

```
[고객명]님, 안녕하세요 😊

오늘은 비슷한 상황의 고객분 사례 하나 공유드려요.

──────────────────
30대 초반 직장인 A씨
월수입 350만원 / 맞벌이 / 자녀 1명

처음엔 "보험은 나중에" 생각하셨어요.
대신 ISA + 연금저축에만 집중하셨죠.

3년 후, 배우자 입원으로
예상치 못한 비용 1,200만원 발생.
모아둔 자산에서 꺼내야 했습니다.

이후 리뷰 미팅에서 하신 말씀:
"그때 보장 세팅을 같이 했으면 좋았을 것 같아요."
──────────────────

포트폴리오는 수익만큼 방어도 중요합니다.
[고객명]님 상황에 맞는 최적 구성,
다음 미팅에서 함께 확인해요 🙂

남진우 드림
✅ 검토 후 발송해주세요
```

---

## D+3 메시지 (3차 미팅 전날)

**목적:** "결정 자리"가 아님을 프레이밍 → 거부감 제거

```
[고객명]님, 내일 미팅 일정 확인드려요 📅

[날짜/시간/장소]

한 가지만 말씀드리면요.
내일 미팅은 '결정하러 가는 자리'가 아니에요.

지난 두 번의 미팅에서 나눈 내용들
— 재무목표, 포트폴리오 구성, 상품 설명 —
을 한 권의 책자로 정리해서 드리는 자리예요.

보시면서 궁금한 점 여쭤보시고,
마음에 드시면 진행하시고,
더 생각해보고 싶으시면 그래도 됩니다.

편하게 오세요 😊
내일 뵙겠습니다.

남진우 드림
✅ 검토 후 발송해주세요
```

---

## 작성 원칙

1. **판매 언어 금지** — "가입하세요", "놓치지 마세요" 없음
2. **교육 + 공감 톤** — 고객 편에서 설명하는 느낌
3. **짧게** — 모바일에서 스크롤 없이 읽히는 길이
4. **고객명 반드시 포함** — 개인화 필수
5. **발송 전 진우님 검토** — 수치/날짜는 반드시 확인
