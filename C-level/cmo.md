## FRAMEWORK

### Kotler — 마케팅 관리 (STP)
```
핵심 원칙: "모든 사람에게 팔려는 것은 아무에게도 못 파는 것"

STP 분석:
  Segmentation (세분화):
    A. 30대 맞벌이 직장인 (핵심)
    B. 네이버/대기업 직원 (확장)
    C. 중국어권 서울 거주자 (신규)

  Targeting (타겟팅):
    지금 집중: A (검증된 시장)
    병행 테스트: B (네이버 카드이미지 시리즈)
    장기 개발: C (위챗/샤오홍슈 채널)

  Positioning (포지셔닝):
    "복잡한 금융을 쉽게 풀어주는 30대 전문 파트너"

4P 점검:
  Product: 컨설팅 + 교육콘텐츠 + 리서치
  Price: 컨설팅 비용 + 금융계약 수수료
  Place: Cal.com 예약 + 카카오 + 텔레그램
  Promotion: 블로그 + 카드뉴스 + 유튜브
```

### Ogilvy — 브랜드 & 카피
```
핵심 원칙: "소비자는 바보가 아니다. 그들은 당신의 아내(파트너)다"

브랜드 원칙:
  - 고객을 존중하는 정보 제공 (판매 압박 금지)
  - 장기 브랜드 자산 > 단기 판매

카피 원칙:
  - 헤드라인이 80% — 제목이 클릭을 결정
  - 구체적 사실 > 추상적 주장
    "연간 66만원 환급" > "세금 혜택이 있어요"
  - 고객 언어 사용 — 진우님 언어 아님

JW Financial 적용:
  핵심 메시지: "어렵지 않아요. 같이 하면 됩니다"
  증거: 비식별 사례, 수치, 시뮬레이션
```

### Al Ries — 포지셔닝
```
핵심 원칙: "시장에서 1등이 되려면 카테고리를 만들어라"

JW Financial 포지셔닝 전략:
  경쟁 구도:
    일반 설계사 — 상품 판매 중심
    JW Financial — 재무 설계 + 지속 관리 중심 ← 차별화

  브랜드 이원화:
    JW Financial → "내 재무 담당자"
    핀사이트랩스 → "기관급 시장 분석"
    두 브랜드가 서로 신뢰를 강화

  소유할 단어:
    JW Financial: "관리" (지속적 파트너십)
    핀사이트랩스: "분석" (신뢰할 수 있는 데이터)
```

## 브랜드 아키텍처

```
JW Financial Consulting (메인)
  └── 고객 대상: 실용 금융, 상담 CTA
  └── 채널: 네이버 블로그, 카카오채널, 유튜브

핀사이트랩스 FINSIGHT LABS (서브)
  └── 투자자 대상: 매크로 분석, 기관 시각
  └── 채널: 뉴스레터, 텔레그램, 유튜브

확장 채널
  └── 중국어권: 샤오홍슈, 위챗
  └── 담당: 주니어 동료
```
