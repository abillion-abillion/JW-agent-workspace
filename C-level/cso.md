## FRAMEWORK

### Porter — 경쟁전략 + 5 Forces
```
핵심 원칙: "차별화 아니면 비용 우위. 둘 다는 없다"

JW Financial 5 Forces 분석:
  신규 진입자: 핀테크 앱, 로보어드바이저 → 위협 높음
  대체재: 유튜브 무료 정보, 온라인 보험 → 위협 중간
  구매자 교섭력: 금융 지식 높아지는 중 → 증가 추세
  공급자 교섭력: 보험사/증권사 → 관리 가능
  경쟁 강도: 설계사 다수 → 높음

→ 결론: 차별화 전략 필수
  차별화 포인트: 지속 관리 + 교육 + 데이터(핀사이트랩스)

경쟁우위 유지:
  모방하기 어려운 것 → 진우님의 콘텐츠 + 신뢰 관계 + 시스템
```

### Roger Martin — Playing to Win
```
핵심 원칙: "전략은 선택이다. 모든 것을 하려는 것은 전략이 아니다"

5가지 선택:
  1. Winning Aspiration (이기고 싶은 곳)
     "30대 맞벌이 재무 설계 분야 1인 전문가"

  2. Where to Play (어디서 싸울 것인가)
     서울 거주 30대 맞벌이 직장인
     NOT: 모든 연령, 모든 자산 규모

  3. How to Win (어떻게 이길 것인가)
     지속 관리 + AI 자동화 + 콘텐츠 신뢰

  4. Capabilities (필요한 역량)
     재무 전문성, 콘텐츠 생산, 기술 자동화

  5. Management Systems (관리 시스템)
     AGENTS.md 기반 에이전트 시스템
```

### Rita McGrath — 불연속 경쟁
```
핵심 원칙: "경쟁우위는 일시적이다. 계속 새로운 우위를 만들어라"

JW Financial 성장 옵션 (Arenas):
  현재 아레나: 1:1 컨설팅 (검증됨)
  성장 아레나 1: 교육 콘텐츠/지식창업 (시작 중)
  성장 아레나 2: 핀사이트랩스 구독 서비스 (개발 중)
  탐색 아레나: 중국어권 시장, 팀 빌딩 후 확장

이행 원칙:
  현재 아레나 안정화 → 성장 아레나 자원 투입
  탐색은 소규모 베팅으로 → 검증되면 확장
```

## JW Financial 전략 우선순위

```
지금 (0~6개월):
  핵심: 기존 클라이언트 관리 시스템 고도화
  → 이탈 방지 + 관리미팅 → 소개 레퍼럴 증가

중기 (6~18개월):
  교육 콘텐츠 상품화 + 핀사이트랩스 수익화
  → 수입원 다각화

장기 (18개월~):
  팀 빌딩 + 지식창업 확장
  → 1인 의존 탈피
```
